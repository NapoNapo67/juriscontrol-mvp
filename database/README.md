# 📊 DATABASE

Archivos SQL para la base de datos PostgreSQL (Supabase).

## Archivos

1. **schema.sql** - Crear tablas, índices y funciones
2. **seed.sql** - Cargar datos iniciales (prueba)
3. **triggers.sql** (FASE 6) - Triggers para automatización

## Instalación

Ver: `/docs/GUIA_INSTALACION.md` - PASO 5

1. Copiar contenido de `schema.sql`
2. Pegar en Supabase SQL Editor
3. Ejecutar
4. Repetir con `seed.sql`
5. Repetir con `triggers.sql`

## Contenido

### schema.sql
- 6 ENUMs (estados, tipos)
- 7 tablas organizacionales
- 8 catálogos fijos
- 11 tablas operativas
- Índices y FK

### seed.sql
- 10 tipos de juicio
- 5 tipos de crédito
- 7 etapas procesales
- 10 tipos de gasto
- 3 asistentes de prueba
- Usuario demo

### triggers.sql
- 5 SQL triggers
- Integración con IA y notificaciones
- RLS policies mejoradas
